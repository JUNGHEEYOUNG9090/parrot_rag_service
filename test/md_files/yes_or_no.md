# 앵무새에게 '예'와 '아니오' 의사소통을 가르치는 방법

## 1. '예'와 '아니오' 소통의 힘

- 보호자가 모든 것을 선택하는 대신, 앵무새가 직접 자신의 의사를 표현하게 하는 것은 깊은 유대감을 형성합니다.
- 앵무새가 자신이 원하는 간식, 음악, 활동, 친구 등을 스스로 선택할 수 있게 되면 보호자와의 의사소통 장벽이 허물어집니다.
- 이는 마법 같은 일이 아니라 반복적인 훈련을 통해 가능하며, 보호자와 반려조 모두에게 말로 다 할 수 없는 큰 기쁨을 줍니다.

## 2. '예'와 '아니오' 도구 선택하기

- 앵무새가 두 가지 다른 물체를 구분하여 터치하도록 가르칩니다.
- '예'를 나타내는 도구(예: 초록색 카드)와 '아니오'를 나타내는 도구(예: 빨간색 카드)를 정합니다.
- 색상은 앵무새가 구별하기 쉽고, '초록색은 가라(Go)', '빨간색은 멈춰라(Stop)'와 같이 보호자도 기억하기 쉬운 색을 추천합니다.
- 카드 대신 서로 다른 모양의 물체를 사용할 수도 있습니다.

## 3. 학습 물체와 보상물 준비

- 훈련을 위해 두 가지 어휘 도구가 추가로 필요합니다.
  - 앵무새가 원하는 물체 (예: 간식)
  - 앵무새가 원하지 않는 중립적인 물체 (예: 차가운 물그릇 - 앵무새가 싫어하거나 관심 없는 것)
- 훈련 단계에서 사용할 긍정적 강화물(보상)을 준비합니다.

## 4. 훈련 단계

### 1단계: '예'와 '아니오' 구분하기

- '예' 도구를 새 근처에 제시하고 "예를 터치해!"라고 말합니다. 터치하면 즉시 클릭하고 보상을 줍니다.
- 새가 도구를 터치하도록 위치를 바꾸어가며 반복합니다.
- '아니오' 도구도 동일한 방식으로 훈련합니다.
- 두 도구를 모두 놓고 '예' 도구를 더 가까이 두어 새가 성공적으로 선택하게 합니다. 점차 '아니오' 도구의 거리를 좁히며 정확도를 높여갑니다.

### 2단계: 어휘 도구 학습

- '예'와 '아니오' 도구를 마스터했다면, '간식'과 '물' 같은 두 가지 어휘 도구를 각각 독립적으로 터치하도록 훈련합니다.
- 각 물체를 터치할 때마다 적절한 보상과 언어적 신호를 연결합니다.

### 3단계: '예'와 '아니오'의 의미 연결하기

- '예' 도구와 '간식'을 짝지어 훈련합니다. "간식 줄까?"라고 물으며 '예'/'아니오' 도구를 제시합니다.
- 새가 '예'를 선택하면 간식을 줍니다. '아니오'를 선택하면 다시 질문합니다.
- '아니오' 도구는 '물'과 같은 원치 않는 물체와 짝지어 훈련합니다.
- '아니오'를 선택하면 해당 물체를 치우고 보상을 주어, 거절 의사가 올바르게 수용되었음을 알려줍니다.
